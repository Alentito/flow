
# Research Community Platform

A modern research community web app built with Next.js, TypeScript, and Tailwind CSS.

## Features
- Blog for articles and updates
- Project/model showcase with paper links
- Member authentication (visitor/member)
- Permissions for viewing/editing own or collaborative projects
- Draft mode for unpublished work
- Basic drag-and-drop editor for project/article pages
- Clean, modern UI

## Getting Started

1. Install dependencies:
	```bash
	npm install
	```
2. Run the development server:
	```bash
	npm run dev
	```
3. Open [http://localhost:3000](http://localhost:3000) to view the app.

## Folder Structure
- `/src/app` - Main app routes and pages
- `/src/components` - Reusable UI components
- `/src/styles` - Tailwind and global styles
